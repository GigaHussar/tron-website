# Tron-Inspired Tech Diary

A static, neon-lit digital space to document my technical experiments, tools, ideas, and experiences.  
Inspired by the visual style of the *Tron* movies — glowing cyan (and sometimes orange), dark backgrounds, and a sci-fi terminal feel.

## ✨ About

This site acts as a personal **digital diary** where I think out loud about tech. Just a quiet place to log:

- Things I’ve built (or broken)
- Scripts and bots I’ve experimented with
- Logs from tech tinkering sessions
- Notes on hardware, automation, and programming

## 🛠️ Tech Stack

- **HTML + CSS** only (no JavaScript)
- Static site
- Custom typography using the [Conthrax font](https://www.fontspace.com/conthrax-font-f20776)

## 🎨 Style

The visual design borrows heavily from the *Tron: Legacy* aesthetic:
- Cyan and orange highlights
- Glowing HUD panels
- Angular UI elements and grid overlays
